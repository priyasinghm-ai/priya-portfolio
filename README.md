# Priya Singh Portfolio Website

Production-ready animated React portfolio website for Priya Singh.

## Tech Stack

- React
- Vite
- Framer Motion
- Lucide Icons
- CSS animations

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Create a GitHub repository named `priya-portfolio`
2. Upload this project
3. Import the repository in Vercel
4. Click Deploy
