# Priya Singh Portfolio Website

Modern animated React portfolio website for Priya Singh.

## Tech Stack
- React
- Vite
- Framer Motion
- Lucide Icons
- CSS animations

## Run Locally
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Create a GitHub repository named `priya-portfolio`
2. Upload this project
3. Go to Vercel and import the repository
4. Click Deploy
